# A bot that claims the waifus.

Commands implemented:
* ~echo `message` 
  - Echo command
* ~list
  - Displays claim list if it exists, else generates a base list
* ~set
  - Sets the waifulist
* ~add `waifu`
  - Add waifu to claim list
* \`remove `waifu`
  - Remove waifu from claim list
* ~react `message`
  - React on target messages
